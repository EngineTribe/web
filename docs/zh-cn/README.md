📡 **引擎部落** 项目是 SMM:WE 3.2.x 在线服务器的开源实现，它也被用于引擎部落社区创作的 3.3.0 版本。

---

### ✅ 功能特性

- 👥 原版 Engine-bot 的所有 API
- 🌐 多语言、多客户端和多 IM 支持
- ☁ 本地数据库、OneDrive 或 Discord 存储后端
- 🗄️ 异步 MySQL、PostgreSQL 或 SQLite 数据库支持
